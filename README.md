## 簡介
家裡是經營麵包店的，所以選擇製作以烘焙為主題的電商網站，並作為Vue3前端技術的練習。預計網站功能如下:

  + 首頁
    + 介紹
    + 本日精選 
    + 優惠碼領取
  + 商品
    + 商品篩選
  + 登入
  + 購物車

## 使用技術

  + Vue3 (Options API)
  + VueRouter (hash mode)
  + JavaScript(ES6)
  + RWD
  + Bootstrap
  + ESlint(airbnb)

## 使用 Plugin

  + axios
  + Vee-validate
  + mitt
  + vue3-loading-overlay
